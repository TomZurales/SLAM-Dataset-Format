# SLAM Dataset Format (SDF)
## Purpose
Provide a standard for the creation and distribution of visual SLAM datasets.
## Capabilities
### Storage of all common SLAM dataset elements in one archive
* Input Sources
  * Visual
    * Multi-Camera Capable
    * RGBD Capable - Pending
    * Distortion Parameterization
    * Body Transformation
    * Timestamping
  * Inertial
    * Multi-IMU Capable - Pending
    * Noise Parameterization
  * LIDAR - Pending
* Ground Truth Sources
  * Maps
  * Trajectory
### Storage of common SLAM outputs
* Trajectory Estimations
* Generated Maps
### Library for the simple creation and extraction of SDF datasets
### Suite of tools to simplify the creation and analysis of SLAM datasets
